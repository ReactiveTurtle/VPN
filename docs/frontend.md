# Frontend

## Current Implementation

The frontend is an `Angular SPA` located in `src/VpnPortal.Web`.

## Current Pages

- public landing/request page
- account activation page
- user login page
- admin login page
- user dashboard
- admin requests and operations page

## Current User Experience

- Users can submit requests and activate accounts.
- Users can view devices, trusted IPs, pending IP confirmations, and sessions.
- Admins can approve or reject requests and perform operational actions from a single admin page.

## Current Limitations

- Platform-specific config delivery for iOS/Android/Windows/macOS is not yet represented as a complete UX flow.
- The admin UI is functional but still compact and operational rather than polished.
